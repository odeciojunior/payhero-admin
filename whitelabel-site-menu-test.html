Changes Made:

  • Updated config/whitelabel.php to include specific menu item colors:
    • menu-item-bg - Background for active menu items
    • menu-item-active-bg - Background for active menu item links
    • menu-item-hover-bg - Background for hovered menu item links
  • Modified WhitelabelController.php to generate CSS using whitelabel variables:
    • var(--menu-item-bg, var(--primary-dark, var(--primary)))
    • var(--menu-item-active-bg, var(--primary-light, var(--primary)))
    • var(--menu-item-hover-bg, var(--primary-light, var(--primary)))
  • Updated resources/modules/global/css/new-site.css to use CSS variables instead of hardcoded colors
  • Added support for both dark and light menu themes

How It Works:

  • Fallback Chain: Each CSS property has multiple fallbacks to ensure compatibility
  • Dynamic Colors: Each whitelabel client gets their specific menu colors
  • Flexible: Works with both dark sidebar and light sidebar themes
  • Consistent: Maintains the existing design structure while applying brand colors

Try hovering over the menu items to see the hover effects with the whitelabel colors!
